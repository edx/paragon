# useReversedChildren

<SingleComponentStatus
  frontmatter={
    get(props, 'pageContext.frontmatter') ? props.pageContext.frontmatter : null

/>

<p>
  This hook reverses a collection of children in a safe manner, through React.Children.
</p>

##### Basic Usage

```jsx live
function() {
  const Example = ({ children }) => {
    const reversedChildren = useReversedChildren(children);
    return reversedChildren;
  };

  return (
    <Example>
      <div>1</div>
      <div>2</div>
    </Example>
  );
}
```
