# Hyperlink

##### minimal usage

```jsx live=true
<Hyperlink href="https://en.wikipedia.org/wiki/Hyperlink">edX.org</Hyperlink>
```

##### with blank target

```jsx live=true
<Hyperlink destination="https://www.edx.org" target="_blank">edX.org</Hyperlink>
```

##### with onClick

```jsx live=true
<Hyperlink
  destination="https://www.edx.org"
  target="_blank"
  onClick={(e) => {
    e.preventDefault();
    console.log('click');
  }}
>
  edX.org
</Hyperlink>
```

##### with icon

```jsx live=true
<Hyperlink destination="https://www.edx.org">
  <Icon
    id="SampleIcon"
    className="fa fa-book"
    screenReaderText="Visit edX Home"
  />
</Hyperlink>
```
