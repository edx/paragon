##### basic usage

```jsx live=true
<Pagination
  paginationLabel="pagination navigation"
  pageCount={20}
  onPageSelect={() => console.log('page selected')}
/>
```

##### with initial page selected

```jsx live=true
<Pagination
  paginationLabel="pagination navigation"
  pageCount={20}
  currentPage={15}
  onPageSelect={() => console.log('page selected')}
/>
```

##### with max pages displayed

```jsx live=true
<Pagination
  paginationLabel="pagination navigation"
  pageCount={20}
  onPageSelect={() => console.log('page selected')}
  maxPagesDisplayed={9}
/>
```
