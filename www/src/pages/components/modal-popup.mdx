## ModalPopup

A generic component for creating accessibile modal popup objects.

```jsx live
() => {
  const [isOpen, open, close] = useToggle(false);
  const target = React.useRef(null);

  return (
    <>
      <Button ref={target} variant="outline-primary" onClick={open}>Open modal popup</Button>
      <ModalPopup positionRef={target} isOpen={isOpen} close={close}>
        <div className="bg-white p-3 rounded shadow">
          <p>This could be a menu or tray.</p>

          <Button variant="primary" className="mr-2">Action</Button>
          <ModalCloseButton variant="outline-primary">Close</ModalCloseButton>
        </div>
      </ModalPopup>
    </>
  )
}
```
