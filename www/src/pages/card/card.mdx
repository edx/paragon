##### Basic Usage

```jsx live
<Card style={{ width: '18rem' }}>
  <Card.Img variant="top" src="https://source.unsplash.com/400x200/?nature,flower" />
  <Card.Header>
    <Card.Header.Title>
      Card title
    </Card.Header.Title>
  </Card.Header>
  <Card.Section>
    <Card.Section.Content>
      <Card.Section.Text>
        This is a card section. It can contain anything but usually text, a list, or list of links. Multiple sections have a card divider between them.
      </Card.Section.Text>
    </Card.Section.Content>
  </Card.Section>
  <Card.Footer>
    <Card.Footer.Actions>
      <Card.Footer.Action as={Button}>
        Some action
      </Card.Footer.Action>
      <Card.Footer.Action href="https://edx.org" className="btn-tertiary">
        Another action
      </Card.Footer.Action>
    </Card.Footer.Actions>
  </Card.Footer>
</Card>
```

##### With Full Card Click

```jsx live
<Card style={{ width: '18rem' }}>
  <Card.Header>
    <Card.Header.Title>
      <a href="#" className="full-card-click">Card title</a>
    </Card.Header.Title>
  </Card.Header>
  <Card.Section>
    <Card.Section.Content>
      <Card.Section.Text>
        This is a card section. It can contain anything but usually text, a list, or list of links. Multiple sections have a card divider between them.
      </Card.Section.Text>
    </Card.Section.Content>
  </Card.Section>
  <Card.Footer>
    <Card.Footer.Actions>

  </Card.Footer>
</Card>
```
