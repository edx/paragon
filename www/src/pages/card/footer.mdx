### Card.Footer

<p>
  Note: This component is responsive based on its own element width, not the screen size. See <a href="/hooks/useResizeDetector">useResizeDetector</a> for more details.
</p>

##### Basic Usage

```jsx live
<Card>
  <Card.Footer>
    <Card.Footer.Content>
      Footer Text
    </Card.Footer.Content>
    <Card.Footer.Actions>
        <Card.Footer.Action href="https://edx.org" className="btn-primary">
          Action 1
        </Card.Footer.Action>
        <Card.Footer.Action href="https://edx.org" className="btn-tertiary">
          Action 2
        </Card.Footer.Action>
      </Card.Footer.Actions>
  </Card.Footer>
</Card>
```

##### With Custom Breakpoints

Try reducing the breakpoints.stacked prop from 1000 to something more like 500, and
then shrink the width of your browser until the footer hits your custom breakpoint.

```jsx live
<Card>
  <Card.Footer
    breakpoints={{
      stacked: 1000,
    }}
  >
    <Card.Footer.Content>
      Footer Text
    </Card.Footer.Content>
    <Card.Footer.Actions>
        <Card.Footer.Action as={Button}>
          Action 1
        </Card.Footer.Action>
        <Card.Footer.Action as={Button} variant="tertiary">
          Action 2
        </Card.Footer.Action>
      </Card.Footer.Actions>
  </Card.Footer>
</Card>
```
